⌨ Controls & Keybinds

Drive Modes (click buttons in control bar)

DISARMEDAll thruster commands zeroed — ROV won't move. Safe state.
DRIVE / ARMEDFull manual control via gamepad. Stabilization flags controlled independently.
STABILIZEManual control + auto pitch/roll correction from IMU. Enables STAB flag automatically.

Keyboard Shortcuts (work on Control screen)

SToggle pitch/roll stabilization ON/OFF
DToggle depth hold ON/OFF
YToggle yaw hold ON/OFF
↑ ArrowIncrease gain by 10%
↓ ArrowDecrease gain by 10%
ESCEmergency stop — sets DISARMED mode and clears all flags

Gamepad Axes (Axis Layout: ORIGINAL)

Left Stick YVertical / Heave — up = ascend, down = descend
Right Stick YForward / Backward
Left Stick XYaw (rotate) — left = turn left, right = turn right
Right Stick XLateral strafe — left = strafe left, right = strafe right

Axis layout matches thrust_sender.py CONTROL_LAYOUT setting.

Gamepad Buttons

Button 9Toggle stabilization (same as S key) — typically Start/Menu button
D-pad UpIncrease gain by 10%
D-pad DownDecrease gain by 10%

Button numbering is browser Gamepad API index — may vary by controller. Check browser console for button debug output when pressing buttons.

Control Parameters

Deadzone5% (axes under 0.05 are treated as zero)
Gain Range10% – 100%, step 10%, default 100%
Combined Thrust LimitH-group + V-group ≤ 150% (same as thrust_sender.py)
Send Rate50 Hz (matches thrust_sender.py SEND_HZ)
Control Timeout (Pi)0.8 s — Pi zeros thrusters if no packet received

MOSFET Toggle

MOSFET OFFPhysical power cut to servos / accessories. Does NOT disarm thrusters.
MOSFET ONPower enabled. Set mode to ARMED or STABILIZE to move thrusters.
